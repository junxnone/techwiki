# Brief
> 版权法默认禁止共享，也就是说，没有许可证的软件，就等同于保留版权，虽然开源了，用户只能看看源码，不能用，一用就会侵犯版权。所以软件开源的话，必须明确地授予用户开源许可证。
目前，国际公认的开源许可证共有80多种。它们的共同特征是，都允许用户免费地使用、修改、共享源码，但是都有各自的使用条件。


## Open Source License List
根据使用条件的不同，开源许可证分成两大类。
- 宽松式（permissive）许可证
- Copyleft 许可证

### 宽松式许可证
- [ ] BSD
- [ ] Apache
- [ ] MIT

### Copyleft 许可证
- [ ] GPL
- [ ] GPL 2.0
- [ ] AGPL - Affero GPL 
- [ ] LGPL
- [ ] MPL - Mozilla
