## History

- 1985 Richard Stallman starts GCC as GNU C Compiler
- 1991 GCC 1.x => GCC 2.x
- 1997 EGCS fork
- 1999 EGCS becomes official GCC 2.95 with GCC Steering Committee
- 2001 GCC 3.x
- 2005 GCC 4.x then from 6 month to one year release
- 2020 GCC11
